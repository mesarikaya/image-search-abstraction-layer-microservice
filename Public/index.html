API: Image Search Abstraction Layer microservice

This microservice makes a Google search based for the user and also enables listing the latest searches in the database. Below is how to use this service:
    1-) For new search, enter the web link to the explorer address.

      Example:

      https://image-search-microservices.herokuapp.com/search/cats

      Example output:

      {{"img-url":"https://i.ytimg.com/vi/tntOCGkgt98/maxresdefault.jpg","snippet":"Dec 30, 2013 ... Funny Cats Compilation ...", "thumbnail":"https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcQq22iXWNm3ZKR58XXOkyriSN2BnLJhLdymzuY10LjtAR6U91z7U1IRVY4"}, ...}
    2-) For searching the database for the latest entries:

      Example:

      https://image-search-microservices.herokuapp.com/list/latestsearches

      Example output:

      [{"_id":"58d269ec3e76b821835a74a4","search_term":"cats","timestamp":"2017-03-22T12:11:24.521Z"}, {"_id":"58d2696e3e76b821835a74a3","search_term":"cats","timestamp":"2017-03-22T12:09:18.417Z"}, {"_id":"58d268f03e76b821835a74a2","search_term":"test","timestamp":"2017-03-22T12:07:12.099Z"}]
